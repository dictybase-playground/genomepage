## Item key
The `item` key will hold the following object

```js
item: {
    type: ...,
    data: ....
}
```

The different types and example shapes of the data objects are described below.

### simple

```js
    data: 'sadA'
```

### link

```js
    data: {url: 'http://...', text: 'abcd'}
```

### linkImage
```js
    data: {url: 'http://', image: 'http://', text: 'efdk' }
```

### linkProps
```js
    data: {ur}
```
